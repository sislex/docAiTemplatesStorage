# docAIEngine

This is a code bundle for docAIEngine. The original project is available at https://www.figma.com/design/0fZd3wNMv4DwgYvEPwxivU/docAIEngine.

## Running the code

1. Install dependencies:
   `npm i`
2. Configure API URL:
   copy `.env.example` to `.env` (default backend: `http://localhost:4000`)
3. Start the dev server:
   `npm run dev`
