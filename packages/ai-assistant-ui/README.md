# @templateStorage/ai-assistant-ui

Переиспользуемый UI-модуль AI-ассистента для React + MUI.

## Что внутри

- controlled-компонент `AiAssistant`
- reducer и action-типизация для родительского приложения
- расширяемый формат сообщений (`status`, `meta`, `attachments`)
- Storybook-сценарии состояний

## Скрипты

```bash
npm run test --workspace=packages/ai-assistant-ui
npm run build --workspace=packages/ai-assistant-ui
npm run storybook --workspace=packages/ai-assistant-ui
```
